# 🖥️ 局域网部署指南 - AI 视频工厂

## 快速开始（3 步搞定）

### 第 1 步：启动服务器

```powershell
cd c:\Users\Lenovo\WorkBuddy\Claw\ai-video-pipeline
node server.js
```

启动后会显示：
```
🎬 AI 视频工厂服务器已启动
📍 本机访问: http://localhost:3000
🌐 局域网访问: http://192.168.1.100:3000  ← 用这个地址！
```

### 第 2 步：让其他设备访问

同一 WiFi/局域网下的设备（手机、同事电脑），浏览器输入：
```
http://192.168.1.100:3000
```
（IP 地址以你电脑实际显示的为准）

### 第 3 步：配置 API Key（可选）

在项目目录创建 `.env` 文件，填入：
```
DEEPSEEK_API_KEY=你的key
```

## ⚠️ 常见问题

### Q：别人访问不了？
**检查 Windows 防火墙：**
1. 打开「控制面板」→「Windows Defender 防火墙」→「高级设置」
2. 点击「入站规则」→「新建规则」
3. 选择「端口」→ 下一步 → 输入 `3000`
4. 选择「允许连接」→ 全部勾选 → 完成

或者用管理员权限运行 PowerShell 一键放行：
```powershell
New-NetFirewallRule -DisplayName "AI视频工厂" -Direction Inbound -LocalPort 3000 -Protocol TCP -Action Allow
```

### Q：每次重启电脑都要重新启动？
**设为开机自启（Windows）：**

创建一个 `start.bat` 文件：
```bat
@echo off
cd /d C:\Users\Lenovo\WorkBuddy\Claw\ai-video-pipeline
node server.js
pause
```

然后按 `Win+R` → 输入 `shell:startup` → 把 `start.bat` 放进去即可。

### Q：想让外网也能访问？
用内网穿透工具：
- **ngrok**（免费）: `ngrok http 3000`
- **cpolar**（免费）: `cpolar http 3000`
- **花生壳**（免费版）
