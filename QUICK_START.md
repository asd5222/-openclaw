# 🚀 快速启动指南

## 第一步：启动服务器

**方法1 - 快速启动（推荐）：**
```bash
npm start
```

**方法2 - 使用启动脚本（Windows）：**
```bash
start.bat
```

**方法3 - 完整启动（带自动更新功能）：**
```bash
npm run full
```

---

## 第二步：打开浏览器

启动成功后，在浏览器中访问：
```
http://localhost:3000
```

---

## 如果没有数据

看到页面但内容为空？运行以下命令获取数据：

```bash
npm run update-news
```

这会自动抓取最新的AI新闻和视频数据（可能需要2-3分钟）。

---

## 常见问题

### Q1: 服务器启动后没有反应？
**A:** 检查：
- 是否正确安装了依赖：`npm install`
- 3000端口是否被其他程序占用

### Q2: 页面加载很慢？
**A:** 第一次加载需要抓取数据，请耐心等待2-3分钟

### Q3: 如何更新数据？
**A:** 
- 方式1：点击页面右上角的"刷新"按钮
- 方式2：运行 `npm run update-news`

### Q4: 如何停止服务器？
**A:** 在命令行窗口按 `Ctrl + C`

## 文件说明

- `server-simple.js` - 简化版服务器（快速启动）
- `server.js` - 完整版服务器（支持自动更新）
- `start.bat` - Windows启动脚本
- `test.js` - 服务器测试脚本
- `package.json` - 项目配置

---

## 端口被占用？

如果3000端口被占用，可以修改端口：

创建 `.env` 文件：
```env
PORT=8080
```

然后访问 `http://localhost:8080`
